1. 游戏规则
>> 使用给定的四种炮塔布阵，击溃所有的敌人，阻止它们到达地图终点。
>> 敌人会选择一条最短的路径，你不能用炮塔将地图封死。
>> 击杀敌人可以获得金币，放置和升级炮塔都需要金币。
>> 你有20条生命，每当一个敌人成功到达终点，你将损失一条生命。
>> 生命值归零时，游戏结束；挺住30波进攻即获胜。

2. 游戏操作
>> 在开始界面点击任一地图图标开始游戏。
>> 拖拽地图下方的四种炮塔到地图上的合适位置来放置它们。
>> 点击地图上的炮塔可以对它们进行升级或售卖。
>> 点击地图左上角的按钮可以暂停或加速游戏。

3. 炮塔
(1) 机枪塔
>> 价格低廉，拥有小范围的射程，对单体敌人造成中等伤害。
(2) 毒液塔
>> 本身不造成伤害，但会使中等范围内的敌人中毒。
>> 中毒后的敌人将收到更多来自其它炮塔的伤害(1.2倍)。
>> 中毒效果将在敌人走出攻击范围后持续一小段时间。
(3) 火焰塔
>> 对小范围内的群体敌人造成小而连续的火焰伤害。
>> 在Frostbite地图中对寒冰士兵造成额外100%伤害。
(4)雷电塔
>> 价格昂贵，射程范围大，对单体敌人造成巨额伤害。
>> 对敌方飞行目标造成额外50%伤害。
